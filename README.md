# 🏥 MEDILINK

**MEDILINK** is a smart healthcare platform designed to simplify doctor consultations and enhance patient experience using modern technology.

---

## 🚀 Features

- 👨‍⚕️ Doctor Consultation Booking  
  Book appointments with specialists like therapists, dentists, dermatologists, etc.

- 🧠 AI-Based Emotion Detection  
  Detects patient emotions using:
  - Facial expressions  
  - Voice tone  
  - User behavior  

- 📄 Medical Report Analysis  
  Extracts and interprets data from uploaded reports  

- ⚡ Fast & Efficient System  
  Built for real-time interaction and quick responses  

---

## 🛠️ Tech Stack

- **Frontend:** React.js, Tailwind CSS  
- **Backend:** FastAPI (Python)  
- **AI/ML:** PyTorch, Transformers (ViT-based models)  
- **Database:** PostgreSQL  

---

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/Sumukh-161/MEDILINK.git

# Navigate to project folder
cd MEDILINK

# Create virtual environment
python -m venv venv

# Activate environment
venv\Scripts\activate   # Windows

# Install dependencies
pip install -r requirements.txt
```

---

## ▶️ Running the Project

```bash
uvicorn main:app --reload
```

---

## 💻 Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

## 📸 How It Works

1. Capture user data (image, voice, behavior)  
2. AI model analyzes emotional state  
3. Suggests suitable doctor or consultation  
4. Allows booking and report upload
